GraphWithHeuristics 

class GraphWithHeuristics extends AnyRef

A class to represent the graph and implement IDA* algoritm https://en.wikipedia.org/wiki/Iterative_deepening_A*

Specify path to your resources when creating a new GraphWithHeuristics your resources should contain:

Example:

  1. run scala.jps.Main "C:\ida\src\main\resources\graph1"

    Your resources should contain three files defining graph:

    nodes: 

    1
2
3
4
5

    heuristics: 

    200
247
162
72
0

    neighbors: 

    0->1-69,3-135
1->0-69,2-75
2->1-75,3-165
3->0-135,2-165,4-80
4->2-180,3-80

1.0
